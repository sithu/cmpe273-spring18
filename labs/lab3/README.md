### How to Setup

```sh
virtualenv -p python3 my-venv
. my-venv/bin/activate
pip3 install -r requirements.txt
```

### How to run Chat server

```sh
python3 server.py
```

### How to run a Chat client

```sh
python3 client.py [username]
```

__Expected Output__

```sh
Connected to the chat server.
> Hi
```
